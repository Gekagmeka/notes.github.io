Методы вставки элементов в html документ:

.before (вставить перед выбранным элементом)
.after (вставить после выбранного элемента)
.prepend (внутрь нашего элемента, перед всем содержимым)
.append (внутрь нашего элемента, после всего содержимого)

-------------
.insertAdjacentHTML  --> используется для вставки строки с html тегами

.beforebegin (вставка html перед выбранным элементом)
.afterbegin (вставка html внутрь элемента перед всем контентом)
.beforeend (вставка html внутрь элемента после всего контента)
.afterend (вставка html после выбранного элемента)
